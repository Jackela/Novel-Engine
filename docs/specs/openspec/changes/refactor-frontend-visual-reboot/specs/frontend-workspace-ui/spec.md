## ADDED Requirements
### Requirement: Guest entry CTA remains prominent after visual redesign
The landing page MUST preserve a clear, primary “Continue as guest” action even after visual redesign.

#### Scenario: Guest enters from landing page
- **GIVEN** a visitor starts at `/` without an existing session
- **WHEN** they view the primary CTA area
- **THEN** the “Continue as guest” action is visible and styled as a primary call-to-action.
