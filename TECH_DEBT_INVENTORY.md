## 🏢 外包模式工作流 (标准化)

### 1. 任务分配流程
```
管理层 (正式员工)
  ↓ 创建任务
分配任务 → Worker Subagent (外包)
  ↓ 完成提交
提交审查 → QC Agent (内包)
  ↓ 审查结果
  ├─ 通过 → 合并代码
  └─ 返工 → Worker修复
```
