## Codebase Patterns
- Hexagonal architecture: domain -> application -> infrastructure

- Result pattern: Ok(value) and Err(Error(...)) from src/core/result.py
- Route decorators: NO /api prefix (added by app.include_router)
- Frontend: shadcn/ui + Tailwind ONLY, no custom CSS
- Zustand for global state, TanStack Query for server state
- Follow existing patterns in world/ context
